# 🏺 Clay & Craft – Handmade Ceramics E-Commerce

A premium e-commerce product page for a handmade ceramics brand. Built with **Vanilla HTML, CSS & JavaScript**.

![Desktop View](screenshots/desktop.png)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏺 **Handmade Feel** | Warm terracotta & cream color palette |
| 🖼️ **Interactive Gallery** | Product images with thumbnail switching |
| 🎨 **Color Selection** | 4 finish options with visual feedback |
| 🛒 **Add to Cart** | With quantity selector and animation |
| ⭐ **Customer Reviews** | Social proof with star ratings |
| 📬 **Newsletter** | Email subscription section |
| 📱 **Fully Responsive** | Perfect on all devices |
